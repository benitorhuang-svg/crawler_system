# 職位大項目歸類方案 (15 項)

本文件定義了將來自不同求職平台（104、1111、Cakeresume、Yes123）的職位頂層分類，歸納為 15 個大項目歸類。

這些大項目歸類將作為新的父級分類，用於填補資料庫中 `tb_category_source` 表格裡 `parent_source_id` 為 `null` 的記錄。
